# Blogger Application

This is a simple full-stack blogging application built using the MERN stack (MongoDB, Express, React, Node.js).

## Features
- Add blog posts
- View all blog posts

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas

## How to Run
1. Start backend:
   cd backend
   npm install
   node server.js

2. Start frontend:
   cd frontend
   npm install
   npm start
